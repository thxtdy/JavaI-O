[ 실행 흐름 ]
	( 클라이언트 )
	로그인에서 IP, PORT, ID를 입력한 후, 서버와 연결이 되면 접속완료.
	대기실에서는 현재 접속중인 유저와 만들어진 방을 볼 수 있다.
			비밀 메세지를 보낼수 있고, 보낼 상대는 리스트에서 선택해야 한다.
			방을 만들수 있고, 방을 만들고 나면 다른방에는 들어가지 못한다.
	채팅에서는 들어간 방이 없으면 메세지가 아무에게도 가지 않는다.
			내가 들어있는 방에 누군가 입, 퇴장을 하면 알려준다.
			방 나가기를 하면 채팅창은 리셋된다.

	( 서버 )
	서버를 시작하면, 해당 서버로 들어오는 클라이언트측에서 일어나는 상황을 화면에서 확인할 수 있다.
	
[ 기능 ]
	로그인
	채팅창에서 실시간 접속자 리스트, 방 리스트 조회
	방에서 메세지 보내기, 비밀 메세지 보내기 기능
	방 만들기, 방 참여하기
